---

## Features

- Generate a multiplication table for any number.
- Calculate the sum of the multiplication table.
- Interactive loop: practice multiple numbers until you choose to exit.
- Input validation for numeric values.
- Fun and educational for students.

---

## Requirements

- [Dart SDK](https://dart.dev/get-dart) installed on your machine.

---

## How to Run

1. Clone or download this repository.
2. Save the code in a file named `sumplier_madness.dart`.
3. Open your terminal/command prompt.
4. Navigate to the folder containing `sumplier_madness.dart`.
5. Run the program using the following command:

```bash
dart run sumplier_madness.dart
